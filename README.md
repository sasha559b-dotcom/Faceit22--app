# 🎮 CyberLeague Faceit Bot

Telegram-бот для организации внутренних матчей в стиле FACEIT.

## Возможности

- Лобби 5v5 и 2v2
- Система пиков и банов карт
- ELO рейтинг и 7 рангов
- Красивые графические карточки профиля и топа
- Win Streak система
- История матчей
- Панель администратора

## Деплой на Railway

1. Форкни репозиторий
2. Зайди на [railway.app](https://railway.app)
3. New Project → Deploy from GitHub → выбери этот репо
4. В Variables добавь: `BOT_TOKEN` = токен твоего бота
5. Готово!

## Команды

| Команда | Описание |
|---------|----------|
| /start | Главное меню |
| /reg | Регистрация |
| /play5 | Лобби 5v5 |
| /play2 | Лобби 2v2 |
| /stats | Профиль |
| /top | Топ игроков |
| /history | История матчей |
| /ping | Статус бота |

## Админ команды

| Команда | Описание |
|---------|----------|
| /win | Результат матча |
| /cancel | Отменить матч |
| /ban | Забанить |
| /mute | Замьютить |
| /setelo | Установить ELO |
| /admin | Панель управления |
